Kadence - Public API - Examples - Check In To Bookings

This sample application demonstrates how to use the public API for automatically checking in users to bookings using their email address. You can use the logic in this application as a basis to build your own check-in application that integrates with WiFi, door access or other systems. In this example we check to see if a user has a booking today (desk or onsite) and if so, we check them in. If they have multiple bookings we check them into all of them.

This example can be found in ./src/examples/check-in.

You can see more examples back on the main screen.
