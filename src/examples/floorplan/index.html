Kadence - Public API - Examples - Floorplan - Space Availability

This sample application shows how to use the public API to visualise real-time space availability on an office floorplan. It pulls live booking and workspace data to indicate which spaces are currently booked or free. You can use this approach to power kiosk displays, interactive maps, or admin dashboards.

This example can be found in ./src/examples/floorplan.
